# daniil-portfolio-website

My personal, most recently updated portfolio website built with Next.js and TypeScript.
<br />
Technology used in this project include:

- Next.js <br />
- React.js <br />
- TypeScript <br />
- Stylelint (CSS Linting) <br />
- ESLint (JS/TS Linting) <br />
- Tailwind CSS <br />
- Prettier (Code Formatting) <br />
- prettier-plugin-tailwindcss (with a custom config file for full Tailwind v4 Support) <br />
- Webpack (Development) <br />
- Turbopack (Publish) <br />
